# Описание модулей проекта Metal Gear JS
# index.html
Главный HTML-файл, точка входа приложения. 
Содержит: базовую структуру HTML-документа, мета-теги для кодировки и адаптивности, подключение CSS-стилей (style.css), структуру игрового интерфейса: заголовок и подзаголовок, панели информации (уровень, обнаружение, документы), canvas элемент для отрисовки игры, элементы алертов и уведомлений, кнопки управления, инструкции для игрока, футер. Подключение JavaScript модуля (main.js).
# style.css
Стилизация всего пользовательского интерфейса.
Содержит: глобальные сбросы стилей (*, body), стили для тёмной темы с градиентным фоном, стилизацию игровых элементов: заголовков и текста, информационных панелей, кнопок управления с hover-эффектами, сanvas с рамкой и тенью, алертов с анимациями, инструкций в стилизованных блоках. Адаптивные медиа-запросы для мобильных устройств, цветовую схему в стиле Metal Gear.
# main.js
Точка входа JavaScript приложения, инициализация игры.
Содержит: импорт основного класса Game из модулей, обработчик события DOMContentLoaded для безопасной инициализации, получение ссылки на canvas элемент, создание экземпляра игры, назначение обработчиков событий для кнопок управления, запуск игры после полной загрузки страницы.
# modules/Game.js
Основной класс, управляющий всей игровой логикой.
Содержит: инициализацию игровых объектов (игрок, враги, документы), создание и управление уровнями, обработку пользовательского ввода (клавиатура), главный игровой цикл (gameLoop), систему обнаружения и тревоги, логику завершения уровней и игры, механику массового устранения врагов, систему отладки и валидации уровней, управление состоянием игры (playing, detected, levelComplete, gameOver).
# modules/Player.js
Класс, представляющий игрового персонажа.
Содержит: свойства игрока (позиция, размер, скорость, направление), методы обработки движения с учетом коллизий, логику управления (WASD + Shift для бега), систему направлений (вверх, вниз, влево, вправо), визуальное отображение игрока с индикатором направления, отладочную информацию о столкновениях.
# modules/Enemy.js
Класс искусственного интеллекта врагов.
Содержит: систему патрулирования по заданным маршрутам, механизм обнаружения игрока: проверка расстояния, проверка угла обзора (60 градусов), проверка прямой видимости (line of sight). Поведение при тревоге (преследование игрока), логику устранения (только сзади, в слепой зоне 180 градусов), систему усиленной бдительности при массовом устранении, коллизии с другими врагами и стенами, визуальное отображение с зонами видимости.
# modules/Level.js
Класс для описания и отрисовки игровых уровней.
Содержит: структуру данных уровня: массив стен (координаты и размеры), массив врагов с патрульными маршрутами, позицию выхода, стартовую позицию игрока, расположение документов. Метод отрисовки уровня (стены и выход), номер уровня для идентификации.
# modules/Document.js
Класс для игровых объектов - секретных документов.
Содержит: свойства документа (позиция, размер, цвет), состояние (собран/не собран), метод проверки коллизии с игроком, логику сбора документа, визуальное отображение с текстом "DOC" и эффектом блеска.
